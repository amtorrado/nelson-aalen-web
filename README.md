# Nelson-Aalen Web

[![Live](https://img.shields.io/badge/Demo-Live-brightgreen)](https://amtorrado.github.io/nelson-aalen-web/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Herramienta web para análisis de confiabilidad de equipos industriales. 
Calcula el riesgo acumulado de fallas e identifica patrones de desgaste, vida útil o mejora.

**Demo:** https://amtorrado.github.io/nelson-aalen-web/

---

## ¿Qué resuelve?

| Problema | Solución |
|----------|----------|
| ¿Mi equipo está empeorando? | Curva convexa = desgaste |
| ¿El mantenimiento funciona? | Curva cóncava = mejora |
| ¿Es estable la operación? | Línea recta = vida útil |

---

## Características clave

- Sin instalación — 100% navegador
- Sin servidor — datos no salen de tu PC
- Resultados en segundos
- Exporta gráficos PNG

---

## Tecnología

Python (Pyodide) + Plotly.js. Single-page, código abierto.
